## FlixMix
Craft and share movie playlists for your friends

## Installation and Setup Instructions

Installation:
`npm install`

To Start Server: 
`npm run dev-start`

To Compile Front End:
`npm run react-dev`

To Visit Local App:
`localhost:3000`
